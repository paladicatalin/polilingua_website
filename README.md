# PoliLingua Website

Public website for PoliLingua, with multilingual content, careers, service presentation, CV application handling, and private admin tools.

## Stack

- PHP 8+
- MySQL / MariaDB
- Apache / XAMPP compatible
- HTML, CSS, JavaScript

## Main Areas

- Public homepage and careers page
- Multilingual RO / RU / EN content
- Admin-managed services, jobs, and applications
- CV upload handling
